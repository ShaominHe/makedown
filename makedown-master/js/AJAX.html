console.log打印出来的数据，前面没有箭头表示是字符串，有箭头是对象

不需要刷新页面就可以加载后台数据的技术：AJAX
AJAX 全称：async javascript and XML,是一个异步执行的和后台交互的技术，例如网易163的注册

AJAX书写

1、创建ajax对象
var ajax = new XMLHttpRequest();
2、建立一个http请求
ajax .open(“请求方式”，”请求地址”，ture)
参数1：请求方式—get/post
参数2：请求地址—-url
参数3：turn是异步请求，false是同步请求，一般不用同步
3、检测响应是否成功
ajax.readyState代表ajax发送请求的过程：0,1,2,3,4,（状态改变事件）

ajax.onreadystatechange=function(){
if(ajax.status = = 200 && ajax.readyState = = 4){
ajax.responseText; //从服务器返回的文本信息
}
}
4、发送请求
如果是post请求的话，需要在发送请求前，设置post请求头，模拟表单来传递数据给服务器（否贼获取不到send中的参数）
ajax.setRequestHeader(“content-type”,”application/x-www-form-urlencoded”)
ajax.send(“键=值&键=值&键=值…”)

如果是get ajax.send(null)，如果get要发送请求带参数的话在ajax.open的第二个参数地址中加？加发送的数据的键值
