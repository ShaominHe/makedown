五、运算
1、NaN 表示一个非数字，在JS中，当发现无法进行运算时，JS不会报错，而是会返回一个NaN
例：var a =”abc”;
var b =2;
var c = a - b;
console.log(c);
输出的就是NaN
2、赋值运算
=赋值符：左边是变量，右边是值，将右边的值放在左边的变量中
3、算术运算
+求和 -求差 *求积 /求商 %求余

字符串的拼接运算：进行+运算时，如果两边进行的是数字中的加法运算，但是两边只要一边是字符串的话，那就是将两边的值连接成一个，也就是所谓的字符串的拼接
例：var a =12345;
var b =”上山打老虎”
var c =a+b;
console.log(c);
结果是12345上山打老虎
具体情况：数字+字符串，字符串+数字，字符串+字符串
4、比较运算（比较的结果是布尔型）
<,>,>=,<=，= =，= = =

= = =先判断类型是否相同，如果相同了，在比较值是否相等

六、强制转换
1、强制转换为数字类型Number
数字类的字符串，可以转换为具体数字，别的字符串转为数字都是NaN。
布尔型转为数字：true是1，false是0。
undefined直接赋值也是undefined，undefined转数字也是NaN
null转数字是0
2、字符串的转换 String
其他类型转字符串，只是加个引号而已
3、转布尔型 Boolean
数字转布尔型，0是false，其他数字都是true
空字符串转布尔型为false,其他都是true
undefined和null转布尔都是false
4、隐形的转换
数字运算（出了加法）可以悄悄的将数据装成数字
加法：只要有一个是字符串，另一个悄悄转换为字符串
比较运算的时候，会将别的类型转成数字作比较

十三、switch多路判断

var num = parseInt(prompt("请输入一个0~7的数字："));

switch(num){

case 1:

    console.log("今天星期一");

    break; // 停止执行这个switch

case 2:

    console.log("今天星期二");

    break;

case 3:

    console.log("今天星期三");

    break;

case 4:

    console.log("今天星期四");

    break;

case 5:

    console.log("今天星期五");

    break;

default:

    console.log("今天是周末");

}
