进制间的转换

1、10进制转别的进制 toString(要转换为几进制)

var res = num.toString(8); 转为八进制
var res = num.toString(16); 转为十六进制
2、别的进制转10进制 parseInt(被转换的进制数)
var num = 110 ;
var res = parseInt(num,2) 将二进制转为十进制
var res = parseInt(num,8) 将八进制转为十进制
var res = parseInt(num,16) 将十六进制转为十进制
