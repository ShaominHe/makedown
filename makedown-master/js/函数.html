函数

一、函数的预解析
1、函数运行的时候会预解析，将函数和var 提前
2、预解析的时候，如果有函数和变量同名，函数覆盖变量
3、报错会中断代码的执行
4、如果函数有形参，并且函数中有函数声明，
这时候顺序是先给形参赋值，然后函数内部在进行预解析的提升
5、return后面的代码不执行，但是预解析的时候，reutrn后面的var也会被提升
6、匿名函数的定义不会被提升到前面，因为没有名字，提升到最前面也没办法使用
例：console.log(fn)
fn()
var fn=function(){
console.log(123);
}
相当于： var fn;
console.log(fn)
fn()
fn=function(){
console.log(123)
}
7、if条件不成立，代码是不会运行的，但是如果有变量声明，也会被提升

二、变量
全局变量：可以在全局任何地方使用，包括函数内部
局部变量：只能在函数内部使用（私有变量），不能再全局中使用

作用域：
在函数中使用变量：
首先，在函数内部去找这个变量，看看有没有在函数内部定义过——-从局部找局部找不到，就去上一层空间里面找，如果上一层空间也没有，去往上一层的上一层去找。。。直到全局，全局中找到了就使用，找不到就报错，变量 is not defined

三、事件
事件：用户在页面上

事件的三种写法
1、写在标签上，on事件名=“函数调用”，当做标签的属性
2、标签 .onclick=匿名函数
3、标签 .onclick=函数名

事件的种类：
鼠标单击
btn.onclick=function(){
alert(“单击了鼠标”)
}
鼠标双击
btn.ondblclick=function(){
alert(“双击了鼠标”)
}
鼠标放上去
btn.onmouseover=function(){
alert(“鼠标放上去”)
}
鼠标离开
btn.onmouseout=function(){-
alert(“鼠标离开了”)
}
键盘按下去 keydown
window.onkeydown=function(){
alert(“键盘按下去了”)
}
键盘弹起 keyup
window.onkeyup=function(){
alert(“键盘弹起了”)
}
加载事件 load
window.onload=function(){//当页面中所有内容加载完成之后，再来加载本事件
事件
}

表单–提交事件 submit
获取焦点事件 focus
失去焦点事件 blur
内容改变事件 change

四、对象
对象：存储一系列信息的（很多变量组合在一起）
对象定义：
1、var obj={ // 创建了一个空对象
}
以json对象的形式创建的对象
var obj ={
属性1：值1，
属性2：值2，
}
例： var obj ={
name:’张三’，
age:12,
sex:’男’，
height:167,
weight:123,
}

对象的访问
对象.属性名
例：console.log(obj.name)获取对象的值
obj.sex=’女’ 修改对象的值
delete obj.sex 删除对象的值
obj.name=’赵六’ 增加一个属性

五、构造函数
能用new来创建对象的函数叫构造函数–任何函数都可以是构造函数，只不过当使用new来创建对象的时候，这个函数叫做构造函数
