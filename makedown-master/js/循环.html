循环

循环的意义是让代码重复执行
一、循环
1、while循环
while(条件语句){
条件成立执行的代码
}
2、do while 循环
do while不管条件是否成立，先执行一次
3、for循环
for（声明变量并赋初始值；条件表达式；计数器）{
重复执行的代码
}
二、循环的关键字
1、continue，可以跳过当前这次的循环，直接进行下一次的循环
2、break，可终止循环，直接让整个循环结束运行

函数

function 函数名（）{
代码段
}
函数的好处：
1、增强代码的复用性
2、实现模块化的编程
函数的调用：函数名（）
小括号内写参数：形参：定义函数的时候使用，在函数中占位
实参 调用的时候使用，给形参赋值
匿名函数：
function( ){
代码段
}
匿名函数传递参数：
1、将整个函数当成一个整体用括号括起来
(function(a,b){
var c=a+b;
document.write(c)
}(1,2)
2、将匿名函数赋值给变量，调用的时候使用变量名来调用
var fun=function(){
代码段
}
fun()
3、通过事件来调用
注意：
1、函数内部输出一个变量，先从函数内部找，有没有定义过这个变量，如果有，直接输出，如果没有，再从函数外面找
2、函数外部输出一个变量。在函数外部去找
3、函数可以先调用，在定义
4、变量先输出，在定义，结果是undefined
5、只有输出，没有定义，报错

带返回值的函数
return： 终止代码的继续运行
在函数中返回一个内容

三元运算符：
条件？成立时的值：不成立时的值
作用：简化双分支：可以将整个三元运算输出，将整个三元运算赋值给一个变量，三元运算符可以嵌套。不能简化单分支和多分支
也可以进行嵌套
例： var max=a>b? a>c?a:c : b>c?b:c

NaN 类型 number 非数字
isNaN（） 数字 （数字类的字符串）进来返回false，非数字进来返回true
