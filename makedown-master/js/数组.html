数组

一、创建数组
1、var a=[] 空数组
2、var a=new Array() 空数组
var a=new Array(5) 当使用构造函数创建数组的时候，如果小括号放了一个数字5，这个数字表示数组有5个元素，这5个元素是空

使用中括号的方式比较多

二、数组访问
1、定义：简单数组访问，数组中的小空间的编号叫下标/索引，从0开始依次递增。数组的最后一个下标是数组的个数-1

2、给数组添加一个元素，数组可扩容
例： var arr=[1,2,3];
arr[3]=4;
console.log(arr)
结果是arr[1,2,3,4]

3、改变元素的值
例: var arr=[1,2,3]
arr[2]=5;
console.log(arr)
结果是arr[1,2,5]

4、遍历
遍历：将数组中所有元素都访问一遍
数组元素的个数：arr.length
将数组的长度拿到循环外面定义好，能提高循环效率
例：var len=arr.length
for(var i=0;i<len;i++){
arr[i]
}
比较数组中谁最大

三、多维数组
多维数组：数组中包含数组，数组中在包含数组形成了多维度的数组，一般情况下使用的二维数组比较多

四、基础类型和引用类型
数据类型：数字、字符串、布尔、undefined、对象（{ }、数组、null）、函数
简单类型—基础类型：数字、字符串、布尔、undefined、null 存储在栈内存空间的，赋值的时候是将值复制一份给别人—-值传递
复杂类型—引用类型：对象、数组、函数
存储的时候存储在堆内存，将堆内存的地址存储在栈内存。将自己的地址复制一份给别人—两个变量共享一个地址，改变一个，另一个也改变了—址传递

五、数组的方法
1、合并数组：concat
concat返回一个合并后的新数组，原来的arr并没有发生改变
2、给数组添加/删除元素：
（1）push：在数组末尾添加一个元素。
push里面需要一个参数，是要添加的新 元素。push已经将原来的arr改变了
（2）pop：将数组的最后一个元素删掉
不用参数也能将原来的arr改变了
（3）unshift：在数组开头添加一个元素
（4）shift：将数组的第一个元素删掉
（5）splice：给数组删除元素，或者再删除的那个位置添加元素。splice ( start,count ) start这个参数，代表从哪个位置开始删除。count表示删除的元素个数。splice可以有第三个参数：将要在删除的位置添加新的元素
例：var arr= [1,2,3,4]
arr.splice（1,2）
结果是arr[1,4]
arr.splice(1,2,8)
结果是arr[1,8,4]
(6)sort：将数组中的元素按从小到大的顺序排列
例：var arr=[5,9,6,8,3]
arr.sort（）
console.log（arr）
（7）reverse ：数组翻转
例：var arr=[1,2,3];
arr.reverse();
console.log(arr)
结果是：arr[3,2,1]
（8）使用连接符将数组所有元素拼成一个字符串 join ( ) 需要一个参数，将每个元素连接起来的连接符
例：var arr=[1,2,3];
var str=arr.join(“-“) //返回连接后的字符串
console.log(str);
结果是：a-b-c
（9）将字符串炸开，组成数组 split ( ) 需要一个参数，分隔符
例：var str=’1,2,3’
var arr=str.split(‘,’)
console.log(arr)
结果是：[“1”,”2”,”3”]
（10）instanceof Array 判断这个值是否是数组，是true，否false

六、冒泡排序
冒泡排序：数组中相邻的元素进行比较大小，如果不符合规则（升序/降序），变换位置，直到最后所有按照这个规则排序

七、选择排序
先找做大值，排在最左边，再找第二大的值，往左边靠……已经排好的，不在参与比较
