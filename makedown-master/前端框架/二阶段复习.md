##### 对象

写法
可以将变量名当做属性名，变量的值作为属性值

```
const age=12;
const obj={
    name:"张三",
    sex:"男",
    age       //输出的结果是age:12
}
```

也可以将变量的值作为属性名

```
const age=12;
const obj={
    name:"张三",
    sex:"男",
    [age]:15    //输出的结果是12:15
}
```

对象中的[]的使用是1：特殊字符 2：使用变量

```
const age=12;
const obj={
    name:"张三",
    sex:"男",
    [age]:15,
    "@#%$^&":123
}
const str="@#%$^&"
console.log(obj["@#%$^&"])
console.log(obj[str])
```

##### find()方法

```
// 数组的扩展
const arr = [{a: 1}, {a: 2}, {a: 3, b: 5}, {a: 4, b: 5}];
find:可以查找第一个符合条件的元素。如果找不到返回undefined.
const status = arr.find(function (v,i,arr2) {
     console.log(v,i,arr2);
     return v.b === 5;
 })
const status = arr.find(v => v.b === 5);
console.log(status);

findIndex:查找第一个符合条件元素所在的下标。如果找不到返回-1
const index = arr.findIndex(v=> v.b === 5);
console.log(index)
```

##### 解构赋值

```
// 传值  赋值
const {b,a,c = 5} = {a:1,b:2};
console.log(a,b,c) // 1 2 5
 
const {b,a,c = a} = {a:1,b:2};
console.log(a,b,c) // 1 2 1
 
const [a,b,c,d=8,[e,f]] = [1,2,3,5,[34,56]];
console.log(a,b,c,d,e,f) // 1 2 3 5 34 56

function fn({a,b}) {
    console.log(a,b); //1 2
}
fn({a:1,b:2});
```

##### padStart方法

补全
字符串的扩展

```
const str="我"
console.log(str.padStart(5,"ABC"))  //结果是：ABCA我
第一个参数限制几个数，第二个参数是用什么补全，而且是在前面补全

padEnd()   在后面补全
const str="我"
console.log(str.padStart(5,"ABC"))  //结果是：我ABCA
```
